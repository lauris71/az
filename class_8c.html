get the index, the containing class and corresponding implementation and instance of a property

Searches class and interface definitions recursively for a property. The order is class->interface[0]->superinterfaces->interface[1]...->superclasses If the index is >= 0, the property definition is def_class->props_self[index]

Parameters
klasscurrent class (either the same as implementation or a superclass)
impltype implementation (can be null for static properties)
insttype instance (can be null for static or implementation properties)
keythe property key
def_classresult for the class where the property is defined
sub_implresult for the actual implementation (either impl or sub-implementation, can be null)
sub_instresult for the actual instance (either inst or sub-interface, can be null)
Returns
the property index in def_class
