Get new typecode, allocate and initialize a class structure.

Gets a new typecode, initializes its class to the minimal functional state and then calls class_init, where a more specialized setup (e.g. registering interfaces and fields) should be done. The final setup is performed only after class_init returns (and thus the sub-interfaces and fields are known).
